Subjects Included Application Development and Emerging Technologies

This subject consists of the following activities/projects: 

Activity 2 - UML Diagram Design Project (Online Coffee & Pastry Ordering System)

Online Coffee & Pastry Ordering System is a web-based ordering and management platform designed to streamline cafe operations and enhance the customer takeaway experience. The system allows customers to browse dynamic menus, customize their beverage and bakery selections, and place secure orders for pickup or delivery. For cafe owners and staff, it provides real-time order tracking, inventory alerts, and efficient payment processing, reducing wait times and replacing manual order-taking with a seamless digital workflow.

UML Final Project

Design and Development of an Online Food Ordering and Delivery System for Don Macchiatos is a web-based ordering and logistics management system designed to digitize and scale the food and beverage operations of Don Macchiatos. Modeled using Unified Modeling Language (UML) to ensure a robust software architecture, the system enables customers to browse menus, customize orders, and track deliveries in real time. It provides staff with automated order queueing, kitchen workflow management, and driver dispatch tools, optimizing daily operations, minimizing delivery times, and replacing manual coordination with a structured digital infrastructure.











System Integration and Architecture 2 (SIA 2)

This subject consists of the following activities 


Activity 2 - My-List-of-Movies System

This activity is a user-friendly, web-based system that helps film enthusiasts manage and curate their personal movie collections more easily. It allows users to track their watch history, organize films into custom watchlists, and rate or review titles based on their personal viewing preferences. The system also provides intelligent movie recommendations tailored to taste and keeps important cinematic details, such as genres, cast, and release dates, neatly organized. With its simple and intuitive features, the system encourages users to discover new stories, reflect on their favorite films, and improve their overall entertainment experience.

Link on how to access Act2- http://127.0.0.1:8000/items


Activity 3 - Movies CRUD System

This activity is a web-based system designed to simplify the process of managing a digital film database. The system allows users to create new movie entries, view detailed film information, update existing movie data, and delete outdated records in real time. It also helps administrators efficiently organize catalogs, update genres or release details, and maintain an accurate, up-to-date media library, providing a more convenient, organized, and seamless data management experience for users.

Link on how to access Act3 - http://127.0.0.1:8000/movies


Activity 4 - Plantcare Tracker Info System

This activity is a web-based management system designed to help plant enthusiasts monitor, track, and maintain the health of their plants efficiently. The system allows users to log plant details, set up watering and fertilizing schedules, and monitor the growth and health status of each plant over time. It also provides an organized platform for managing plant-care information, offering tailored tips based on plant types, and helping users improve their gardening habits to keep their indoor or outdoor plants thriving and healthy.

Link on how to access Act4 - http://127.0.0.1:8000/form







Activity 5 - Integration App System

This activity is a user-friendly web-based platform designed to connect and synchronize different applications, databases, and software services within an organization. The system allows users to configure data pipelines, map information between platforms, and monitor the flow of data in real time. It also provides an organized, automated environment that reduces the need for manual data entry, helps administrators manage api connections efficiently, and ensures information is consistent across all departments, creating a more cohesive, reliable, and streamlined workflow for the organization.
